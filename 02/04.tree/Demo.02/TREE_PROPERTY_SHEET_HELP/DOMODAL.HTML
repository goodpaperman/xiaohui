CTreePropertySheet::DoModal

int CTreePropertySheet::DoModal();

Return Value

An int value that specifies the value of the nResult parameter that was passed to the CTreePropertySheet::EndDialog member function, which is used to close the property sheet. The return value is -1 if the function could not create the property sheet, or IDABORT if some other error occured.

Remarks

Call this member function to invoke the modal property sheet and return the dialog box result when done.
This member function handles all interaction with the user while the property sheet is active.
