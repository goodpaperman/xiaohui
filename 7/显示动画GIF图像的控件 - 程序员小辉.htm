显示动画GIF图像的控件

http://www.XiaoHui.com 日期: 2002-02-01 14:00

本控件Gif89a用VC 6.0和ATL创建，在使用时不需要任何DLL（包括atl.dll和mfc....dll)。本控件提供7个属性和3个方法来播放动画gif文件：

属性：

Filename - GIF文件名；

Autosize - 图片是否自动改变大小；

Autostart - 是否自动显示图片；

Speed - 延时1/100秒，如果未作设置，此为缺省值；

Embed - 如果存储GIF文件到EXE中为真；

Class - 如果用OleContainer的背景颜色绘制空区域为假；

方法：

Stop - 停止播放；

Start - 开始；

About - 介绍本控件。
